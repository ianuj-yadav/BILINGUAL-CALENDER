# 🌍 Bilingual Calendar

A modern and user-friendly **Bilingual Calendar Application** that displays dates, months, and calendar information in **two languages simultaneously**, making it easier for users to view and understand dates across different linguistic preferences.

---

## 📌 Features

✨ Dual-language calendar interface

📅 Monthly calendar view

🌐 Support for multiple language combinations

🔄 Real-time language switching

📱 Responsive design for desktop and mobile devices

🎨 Clean and intuitive user interface

⚡ Fast and lightweight performance


```bash

```

### Open in browser

Simply open:

```bash

```

or use Live Server in VS Code.
